# Fake Org. website

## How to use the project?

### Step 1: Install packages for both client and server

**Server**
```
cd server
npm install
```
**Client**

```
npm install
```

### Step2: Run both server and client

**Server**

```
// in server folder
npm run dev-start
```

**Client**

```
npm start
```
